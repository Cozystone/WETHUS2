# TIMONEY Flutter Setup (Windows)

## 1) Install Flutter SDK
- https://docs.flutter.dev/get-started/install/windows

## 2) Check environment
```bash
flutter doctor
```

## 3) Android emulator for development
```bash
flutter emulators
flutter emulators --launch <id>
```

## 4) Run app
```bash
flutter pub get
flutter run
```

## iOS note
- iOS-style UI development is possible on Windows.
- Real iOS build/archive requires macOS + Xcode.
