
## 2026-03-18 (PM) — WETHUS UI/기능 결정 및 진행
- 상단바 개편 1차: 로그인 상태에서 로그아웃을 프로필 드롭다운으로 통합. 주황색 프로필 칩(사진+이름+플랜)과 hover 드롭다운(요금제/프로필/로그아웃) 도입.
- 상단바 우측 햄버거 아이콘 추가 및 우측 플로팅 사이드 드로어(초기 DM/알림 진입) 도입.
- 프로필 사진 업로드 UX를 드래그앤드롭(dropzone) 방식으로 전환, `applyProfileImageFile(file)`로 파일 처리 분리.
- founder AI 이미지 상태 메시지 정확화: 실제 로드 성공 시에만 “AI 이미지 생성 완료”, 실패 시 “AI 이미지 생성 실패로 기본 이미지를 적용했습니다.”로 분기.
- founder AI 재생성 체감 개선: 생성 URL에 nonce를 붙여 캐시 충돌 방지.
- 탐색 IA 대개편:
  - 기존 탐색 보존 백업: `explore_v1.html` (WETHUS 1.0 라벨)
  - 신규 게이트웨이 `explore.html`: 4개 카테고리( Math&Sci / Art&Culture / Startup&Business / Society&Law ) 대각선 스택 카드+hover 인터랙션
  - 카테고리 상세 `explore_theme.html`: 기존 탐색 기능 재사용 + 테마 tint 적용
  - devMode에서 `WETHUS 1.0` 탭으로 기존 탐색 즉시 접근 가능하도록 경로 확보
- 플랜별 프로필 칩 디자인 분기 요청 반영: Free(오렌지) / Premium(보라 계열 그라데이션) 방향 적용 시작.
- 사용자의 핵심 제약 재확인: “요청하지 않은 부분 수정 금지”, “정상 기능 훼손 금지”, “변경 후 회귀 확인 필수”.
